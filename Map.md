```
JS中的Map对象
1，js创建map对象

var map = new Map();

 

2.将键值对放入map对象

map.set("key",value)

map.set("key1",value1)

map.set("key2",value2)

 

3.根据key获取map值

map.get(key)

 

4.删除map指定对象

delete map[key]

或

map.delete(key)

 

5.循环遍历map

map.

forEach(function(key){
　　console.log("key",key)  //输出的是map中的value值

})
```
